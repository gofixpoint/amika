# Amika Labs

Amika is a software factory for AI coding agents: it gives teams sandboxed
agents that understand a codebase, connect to project data, run checks, and
open pull requests. See [amika.dev](https://www.amika.dev/) for the product
overview.

Amika Labs is where experimental tools and libraries live before they are ready
for the supported Amika surface. Labs is for contributors and early users who
are comfortable with unstable interfaces.

- Go labs: [`go/labs/`](../../go/labs/)

Everything here is a work in progress and carries **no compatibility
guarantees** — APIs, commands, flags, and entire packages may change or
disappear at any time. Don't depend on it from stable code.

Trying a Labs tool? Start with its doc, then build it from the repo root.
