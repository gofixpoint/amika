# @amika/reviews

A TypeScript/React library for reviewing code: upload one or more `.diff` /
`.patch` files (or pass them as text), browse a file tree, view per-file
diffs, and leave threaded comments at line / file / patch / series scope.
Built on `@pierre/diffs/react` and `@pierre/trees/react`.

This package is a work in progress. See
`plans/2026-05-15-amika-reviews-package.md` at the repo root for the full
build plan.
